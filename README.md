# m431_ap24a_ForgottenLands

## The Product
### Forgotten Lands
* A game 2D RPG about a Wanderer in the Forgotten Lands, solving Quests, fighting off Monsters & heading for the Final Boss, to free the Lands.
* Features
  * Fight Monsters
  * Explore Areas
  * Collect Items
  * Upgrade items
  * Clean UX
